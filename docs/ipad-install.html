Install HVF Trainer Pro on iPad

Educational use only. Not a medical device and not for diagnosis, screening, treatment decisions, or patient records.

GitHub Pages

  1. Upload the repository to GitHub.
  2. Enable GitHub Pages.
  3. Open the published HTTPS URL in iPad Safari.
  4. Tap Share → Add to Home Screen.
  5. Launch from the Home Screen in landscape orientation.

Local testing from macOS

  1. Open Terminal in the project folder.
  2. Run python3 -m http.server 8000.
  3. Find your Mac IP with ipconfig getifaddr en0.
  4. On iPad Safari, open http://YOUR-MAC-IP:8000.

For best PWA/offline behavior, use HTTPS hosting through GitHub Pages before adding to Home Screen.
