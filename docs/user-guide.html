HVF Trainer Pro User Guide

Educational use only. This is not a clinical perimeter or diagnostic system.

Basic workflow

  1. Select 24-2, 30-2, or 10-2.
  2. Select eye and strategy.
  3. Choose a simulated defect.
  4. Adjust patient age, response sensitivity, fatigue, and false-positive tendency.
  5. Press Start test.
  6. Maintain fixation on the central mark.
  7. Respond only when a white stimulus is seen.

Inputs

  • Tap the large patient response button.
  • Tap/click the field.
  • Press Space or Enter.
  • Use a Bluetooth switch that sends Space or Enter.

Reports

After the test, review reliability indices, maps, and the simulated report. Use Print/PDF, Copy report, or Download JSON.

Limitations

The simulator does not implement proprietary Humphrey algorithms, calibrated clinical luminance, validated normative databases, or medical-device controls.
